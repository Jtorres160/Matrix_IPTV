# Matrix IPTV Engineering Documentation

Welcome to the Matrix IPTV engineering handbook.

This documentation defines the architecture, engineering standards, and long-term direction of the application.

The purpose of these documents is to preserve architectural consistency as the application grows.

Matrix IPTV is designed as a premium desktop media application built with Electron, React, and modern desktop application principles.

The goal is not to create a website inside a desktop window.

The goal is to create a native-feeling Windows media experience comparable to:

- Plex Desktop
- Jellyfin Desktop
- Spotify Desktop
- Windows 11 applications

---

# Project Vision

Matrix IPTV is a high-performance IPTV and media management application.

The application should provide:

- Fast channel browsing
- Reliable playback
- Large playlist support
- Modern EPG experience
- Multi-user profiles
- Personalized viewing
- Extensible media sources
- Native desktop performance

---

# Core Philosophy

Matrix IPTV follows a layered architecture.
